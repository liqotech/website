# liqo.io new website

Small guide to run and build the project.

## Requirements

- Node.js 20+ (recommended)
- [pnpm](https://pnpm.io/)

## Install dependencies

```bash
pnpm install
```

## Run in development

```bash
pnpm dev
```

This starts the Vite dev server.

## Build for production

```bash
pnpm build
```

The production output is generated in the `build/` folder.
